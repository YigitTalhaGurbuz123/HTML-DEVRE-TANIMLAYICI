Hoşgeldiniz!!

Bu site sayesinde Devreler elemanları hakkında yeterince bilgiye sahip olacaksınız 😎

Özellikler:

  • Site, Python, Html, Css ile geliştirildi.
  • Sitenin devre tanıma bölümü, 30 sınıftan oluşan ve Google Teachable Machine ile eğitilen bir görsel makine ile eğitildi.
  • Site sizin devre elemanı arayarak onun özelliklerini, bağlantılarını ve varsa pin ve portlarını anlatma şansı sunuyor.
  • Sitede Kayıt oluşturup ardından diğer kullanıcılarla sohbet edip sorularınızı onlarada sorabilirsiniz
  • Ayrıca sitede kendi notlarınızı kaydedebilirsinz!
  • Site GNU General Public License v3.0 ile lisanslanmıştır

Sitenin kodlarını canlı deneycekseniz:

Gereken Uygulamalar:

  • Python 3.10 (Çünkü tensorflow bir tek 3.10 da çalışıyor)
  • Visual Studio Code veya herhangi bir IDE

Gereken Kütüphaneler (PIP olarak)

  • pip install tensorflow==2.14.0
  • pip install keras==2.14.0 (yüksek ihtimalle tensorflow ile beraber kurulacak ama yinede deneyin!)
  • pip install numpy==1.24.3
  • pip install Pillow
  • pip install Flask
  • pip install flask-sqlalchemy
  • Sonradan gerek duyarsanız gereken kütüphaneleri requirements.txt içerisinde bulabilirsiniz
